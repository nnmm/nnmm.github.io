    pub i_max: i8,
    pub j_max: i8,
}
Expand description

A 2D area represented by a min + max coordinate pair.

The two coordinates form an inclusive 2D range, i.e. unlike in a half-open range, it’s possible for a point with i == i_max to be contained in the area.
